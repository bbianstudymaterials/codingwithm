C Programming Definitions

CONTROL STATEMENT IN C

  1. if-else statement- It is used to execute a statement block or a single statement depending on the value of a condition.
  2. Switch Statement- It is used to execute a block of statement on the value of a variable or expression.
  3. Break statement- It is a statement which will transfer the control to the end of the loop or statement and execute the program.

LOOP CONTROL STRUCTURES

  1. For Loop- A for loop is used to execute and repeat a block of statement depending on a condition.It has the following form:
    for(initialization;condition;increment)
    {
    statement block;
    }
  2. While loop- A While Loop is used to execute and repeat a block of statement depending on a condition. It has the following form
    while(condition)
    {
    statement block;
    }
  3. Do While Loop- A do-while loop is used to execute and repeat a block of statement depending on a condition. It has the following form.
    do
    {
    statement block;
    }

COMPARISION OF THE LOOP CONTROL STRUCTURES

for loop while loop do-while loop
1 A for loop is used to execute and repeat a statement block depending on a condtion which is evaluated at the beginning of the loop. A while loop is used to execute and repeat a statement block depending on a condition which is evaluated at the beginning of the loop. A do-While loop is used to execute and repeat a statement block depending on a condition which is evaulated at the end of the loop.
2 A variable value is initialized at the beginning of the loop and is used in the condition. A variable value is initialized at the beginning or before the loop and is used in the condition. A variable value is initialized before the loop or assigned inside the loop and is used in the condition.
3 A statement to change the value of the condtion or to increment the value of the variable is given at the beginning of the loop. A statement to change the condition or to increment of the variable is given inside the loop. A statement to change the value of the condtion or to increment the value of the variable is given inside the loop.
4 The statement block will not be executed when the value of the condition is false. The statement block will not be executed when the value of the condition is false. The statement block will be executed once even if the condition is false.
5 A for loop is commonly used by many programmers. A while loop is also commonly used by many programmers. A do-while loop is used in some cases where some condition need to be checked at the end of the loop.

goto STATEMENT

The goto statement is an unconditional transfer of control statement. It is used to transer the condition from one part of the program to another. The place to which the control is transferred is identified a statement label.

Break statement

The break statement is used to transfer the control to the end of the staement block in a loop. It is unavoidable statement to transfer the control to the end of a switch statement after executing any one statement block.


Continue STATEMENT

The continue statement is used to transfer the control to the beginning of a statement block in a loop.


Exit() function

The exit() function is used to transfer the control to the end of a program. It uses one argument in () and the value is zero for normal termination or non-zero for abnormal termination.


Arrays and Subcripted variables.

Arrays are used to store multiple value in a single variable, instead of declaring separate variables for each value.


One-dimensional Array

One dimensional array is a group of elements having the same datatype which are stored under a sinle variable name.

Two dimensional Array

Two dimensional array are the simplest form of the multidimensional array where the data is stored in multiple rows and columns.

String Manipulation in C

An array of characters is known as a string.

String handling function in C

There are four important string handling function in C language.

  1. strlen() function- It is used to find the length of a character string.
  2. strcpy() function- It is used to copy a character string to a character variable.
  3. strcat() function- It is used to join character string.
  4. strcmp() function- It is used to compare two character strings.It return 0 when two strings are identical, otherwise it returns a numerical value which is the difference in ASCII values of the first mismatching characters of the strings being compared.
  5. atoi() function- It is used to convert a string of digits to the integer value.

STRUCTURES and UNIONS

Structures

A Structure is a user defined compound data type which consist of data members related to a person or an item. It is refered to record in other programming languges. A Structure has the following general form.
struct tag-name
{
type data_member1;
type data_member2;
..................
..................
};

Note:

  1. The Structure definition ends with a semicolon(;) after the closing brace.
  2. The Data members of a structure occupy a contiguous block of memory.

Assigning Value of a structure to another structure

All values available in a structure can be assigned to another structure of the same type if all data members in the structures are of the same data type.

Unions

Similarly to a structure, a union also consist of data members but only one data member is active at a time. It has the following form.

union tag_name
{
type data_member1;
type data_member2;
.................
..................
} ;
where type is the data type of the data_members,
data_member1,data_member2.... are the variables,
tag_name is the name of the union.
